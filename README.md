# 📚 LBR MG - Library Management System

A simple **Command-Line Library Management System** built using Python and SQLite.  
This project demonstrates core concepts of **OOP, database handling, and CLI application design**.

---

## 🚀 Features

- 📖 Add new books (title & author)
- 📚 Display all books in the library
- 📕 Issue books
- 📗 Return books
- 🔎 Search books (supports partial matching)
- 📊 View library statistics
- 💾 Persistent storage using SQLite
- ✅ Input validation and error handling

---

## 🛠️ Tech Stack

- Python 3
- SQLite3 (built-in database)
